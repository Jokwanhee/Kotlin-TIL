# 상속
- 자식 클래스를 만들 때 상위 클래스(부모 클래스)의 속성과 기능을 물려받아 계승
- 상위(부모) 클래스의 프로퍼티와 메서드가 자식에 적용됨

### open 키워드를 통한 선언
```kotlin
open class 기반(상위, 부모) 클래스명 {
    ...
}
class 파생(하위, 자식) 클래스명 : 기반 클래스명() {
    ...
}
```
> 코틀린의 모든 클래스는 묵시적으로 Any로부터 상속되어있음.
```kotlin
open class Car() {
    ...
}

class K5(): Car() {
    ...
}
```
